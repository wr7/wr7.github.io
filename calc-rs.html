About the calc-rs project

This website includes images, videos, schematics, and the source code for the calc-rs project.

The calc-rs project can be built and run in two different ways:

  1. It can be similated by the calc-simulator crate on either Windows or Linux.
  2. The calc-rs crate can be compiled and ran on the physical calculator described in the schematics.
